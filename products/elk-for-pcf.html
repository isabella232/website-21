ELK-for-PCF tile

Overview

The ELK-for-PCF Tile is a commercially licensed and supported extension for the Pivotal CF PaaS that installs and manages a tightly interated Elastic ELK cluster that provides multi-tenanted log analysis services for both the PaaS components and the apps deployed to the PaaS

Features

  • Single Sign On: Integration with Cloud Foundry UAA to restrict access to Kibana
  • Multi-tenant access: restrict access to only view logs related to apps in spaces the authenticated user is a member of
  • Environment Setup: Automatically configured Elastic ELK cluster with Queue to handle spikes in log load
  • Ingestion, parsing and enrichment of Elastic Runtime Firehose for App logs, and Elastic Runtime, Redis, RabbitMQ and MySQL syslog component logs
  • Curated dashboards: Comes with sample dashboards to help CF Operators and CF App owners monitor errors, performance and security

Download

The ELK-for-PCF Tile will be available for download from the Pivotal Network in summer 2015
